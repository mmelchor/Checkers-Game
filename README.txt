Checker Game:
  To run this game from the terminal or shell type the following:
         python3.2 main.py

  Then follow the directions and begin to play the game. Click on a piece that you want to move and then click where you want to move it. If you are eating an opponents piece then click on the piece that you want to eat in order to eat it.




Note: This game was develop by Miguel and Esteban freshman year of colloge. Their are still some bugs on the game that we weren't able to fix do to time but the game is currently being rewritten to fix all bugs and be more efficient.

Also note that all files in this folder are required to run the game.
